# Analysis of IKEA product portfolio and expansion opportunities using Python

The project analyzes IKEA’s global product catalog in order to identify patterns in product structure, pricing strategy, and potential expansion directions. Using Python libraries for statistical analysis, visualization, and machine learning, the study highlights market segments, category concentration, and opportunities for diversification.

DATABASE: https://www.kaggle.com/datasets/ranjithpanicker/ikea-global-product-catalog-and-price-2026/data
